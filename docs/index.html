О сервисе

«Персонал ПВЗ» — программный сервис для организации работы персонала пунктов выдачи заказов. Владелец создаёт точку, приглашает сотрудников и ведёт учёт смен и начислений. Сотрудники входят по номеру телефона с подтверждением SMS-кодом.

В

Владельцам ПВЗ

Расписание, зарплата, аналитика, приглашение сотрудников, управление несколькими точками.

С

Сотрудникам

Личное расписание, финансы, заявки на смены, чат с коллегами и уведомления.

Возможности

Всё необходимое для ежедневной работы ПВЗ — в удобном мобильном интерфейсе.

1
Расписание сменПланирование, обмен сменами, уведомления о изменениях.

Использование SMS

Сервис отправляет SMS с одноразовым кодом подтверждения (OTP) только для входа сотрудников и администраторов по номеру телефона. Рекламные и массовые рассылки не используются. Отправитель: подпись, согласованная в личном кабинете SMS Aero.
