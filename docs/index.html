Персонал ПВЗ

Мобильное приложение для владельцев и сотрудников пунктов выдачи заказов (ПВЗ): расписание смен, учёт часов, начисления, выплаты, внутренний чат и уведомления.

О сервисе

«Персонал ПВЗ» — программный сервис для организации работы персонала ПВЗ. Владелец создаёт точку выдачи, приглашает сотрудников и ведёт учёт смен и начислений. Сотрудники входят в приложение по номеру телефона с подтверждением одноразовым SMS-кодом.

Владельцам ПВЗ Расписание, зарплата, аналитика, приглашение сотрудников, управление несколькими точками.
Сотрудникам Личное расписание, финансы, заявки на смены, чат с коллегами.

Возможности

1
Расписание смен — планирование, обмен сменами, уведомления.
2
Зарплата и выплаты — начисления, авансы, штрафы и премии.
3
Аналитика — статистика по точкам и сотрудникам.
4
Чат и уведомления — связь внутри команды ПВЗ.

Использование SMS

Сервис отправляет SMS с одноразовым кодом подтверждения (OTP) только для входа сотрудников и администраторов по номеру телефона. Рекламные и массовые рассылки не используются. Отправитель: подпись, согласованная в личном кабинете SMS Aero.
